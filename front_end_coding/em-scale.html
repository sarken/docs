Ems

This document is mostly obsolete. We still use ems, but we size the containing element #main to 0.875em and all spaces are recalculated from 14px.

The em is a sliding measure. One em is a distance equal to the type size. In 6 point type, an em is 6 points; in 12 point type an em is 12 points and in 60 point type an em is 60 points. Thus a one em space is proportionately the same in any size.
-Robert Bringhurst

The Archive styles use ems to size text and most elements, and to specify margins and padding.

While users can change their browser's text size to suit their needs, the default size on most web browsers is 16 pixels [1]. So, since an em is equal to the text size, 1em is 16 pixels.

The Archive styles use the traditional scale: 6, 8, 9, 10, 12, 14, 16, 18, 24, 36, 48, 60, 72. The main text on the Archive is 14.

To ensure that all browsers can properly display, resize, and zoom on our text, the first thing we do when working with ems is set the body font size to 100%, or the default font size for the user's web browser:

body {
  font: 100%/1.125 'Lucida Grande', 'Lucida Sans Unicode', 'GNU Unifont', Verdana, Helvetica, sans-serif;
}.

Since we want the main Archive text to be 14 pixels, we divide 14 by 16, determining that we should set the font size to 0.875em:

#main { &nbp; font-size: 0.875em;

The Scale in Ems

  • 72/16 = 4.5em
  • 60/16 = 3.75em
  • 48/16 = 3em
  • 36/16 = 2.25em
  • 24/16 = 1.5em
  • 18/16 = 1.125em
  • 16/16 = 1em
  • 14/16 = 0.875em
  • 12/16 = 0.75em
  • 10/16 = 0.625em
  • 9/16 = 0.5625em
  • 8/16 = 0.5em

Choosing a font-size

  • Remember that font-size is inherited from the parent element so it's easier to target specific selectors rather than their containing blocks (i.e., #sidebar li rather than #sidebar. If you choose to style containing blocks, you will need to recalculate the scale from 14px.
  • Decide where the element you are styling fits in the page hierarchy. Is it greater or lesser than p? Definition terms used as headings probably would do well at 1em. Footnotes would do well at 0.75em.

Margins and Padding

It can be a bit difficult to get your head round using ems for margins and padding, so here are some tips:

  • A single space (like between words) is approximately 0.25em.
  • Our font size in #main is 0.875em/1.286, so margin: 0.643em auto; gives you a single blank line (approximately 9px) between block elements.
  • Choosing values from the scale will automatically make your page look balanced.
