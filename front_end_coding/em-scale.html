Ems

I've dismantled the em scale as of 01/09/11 as our parser is still broken and we can't fix it, so we can't use anything that relies on valid XHTML. This document is therefore mostly obsolete. We still use ems, but we size the containing element #main to 0.875em and all spaces are recalculated from 14px.

The em is a sliding measure. One em is a distance equal to the type size. In 6 point type, an em is 6 points; in 12 point type an em is 12 points and in 60 point type an em is 60 points. Thus a one em space is proportionately the same in any size.
~Robert Bringhurst

The Archive styles use ems to size text, most elements, and as units for margins and padding.

The Archive core styles use the traditional scale: 6, 8, 9, 10, 12, 14, 16, 18, 24, 36, 48, 60, 72.

We set the body at 100%, or default. The default text size on most web browsers is 16px [1]. So 1em is equal to 16 pixels.

body { font: 100% Candara, Corbel, Geneva, Verdana, Tahoma, Helvetica, Arial, sans-serif; }

And then the body paragraph text at 14px by dividing 14 by 16:

body p {font-size: 0.875em;}

The Scale in Ems

  • 72/16 = 4.5em
  • 60/16 = 3.75em
  • 48/16 = 3em
  • 36/16 = 2.25em
  • 24/16 = 1.5em
  • 18/16 = 1.125em
  • 16/16 = 1em
  • 14/16 = 0.875em
  • 12/16 = 0.75em
  • 10/16 = 0.625em
  • 9/16 = 0.5625em
  • 8/16 = 0.5em

Choosing a font-size

  • Remember that font-size is inherited from the parent element so it's easier to target specific selectors rather than their containing blocks (ie #sidebar li rather than #sidebar. If you choose to style containing blocks, you will need to recalculate the scale from 14px.
  • Decide where the element you are styling fits in the page hierarchy. Is it greater or lesser than p? Definition terms used as headings probably would do well at 1em. Footnotes would do well at 0.75em.

Margins and Padding

It can be a bit difficult to get your head round using ems for margins and padding, so here are some tips:

  • A single space (like between words) is approximately 0.25em.
  • Choosing values from the scale will automatically make your page look balanced.
